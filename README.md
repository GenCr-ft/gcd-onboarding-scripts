# DevOps Automation Scripts

This directory contains various scripts used for automating DevOps tasks within the GenCr@ft project. These scripts are intended to improve efficiency, ensure consistency, and reduce manual effort in development and operations workflows.

## Subdirectories

* **`onboarding/`**: Contains scripts specifically designed to automate parts of the onboarding process for new developers and AI Gems. This includes scripts like `gftai_onboarding.sh`, `onboarding_win.ps1`, and `gftai_devops_onboarding_addon.sh`.

*(This list should be updated as other categories of scripts or individual scripts are added directly to this directory.)*

## Usage

Each script or script subdirectory should ideally have its own specific README or clear comments within the script explaining:
* Its purpose.
* Prerequisites for execution.
* How to run it (including any required parameters).
* Expected outcomes and potential side effects.

## Contribution

Contributions to these automation scripts are welcome. Please ensure that:
* Scripts are well-commented and easy to understand.
* They follow any relevant coding standards defined in the `devops-standards/` or `gencraft-studio-handbook/`.
* They are tested to ensure they perform as expected.

For overall guidance on DevOps automation, refer to the main `gencraft-devops-automation/README.md`.
